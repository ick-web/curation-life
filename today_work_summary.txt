[2025.06.02] Curation Life 프로젝트 작업 내용 정리

1. Instagram API 대체 작업
----------------------------
- 문제: Instagram Basic Display API 서비스 중단
- 해결: 네이버와 카카오 API로 대체

적용된 API:
* 네이버 API
  - 블로그 검색: 팝업스토어 관련 블로그 포스트
  - 이미지 검색: 팝업스토어 관련 이미지
* 카카오 API
  - 장소 검색: 현재 운영 중인 팝업스토어 위치
  - 블로그 검색: 추가 리뷰 정보

2. CORS 이슈 해결
----------------------------
- 문제: 클라이언트에서 직접 외부 API 호출 시 CORS 오류 발생
- 해결: Next.js API 라우트 구현
  * /api/search 엔드포인트 생성
  * 서버 사이드에서 외부 API 호출 처리

3. 이미지 도메인 설정
----------------------------
next.config.js에 추가된 도메인:
- www.kopis.or.kr
- search.pstatic.net
- k.kakaocdn.net
- culture.seoul.go.kr

4. UI 개선사항
----------------------------
- 팝업스토어 섹션 구조화:
  * 현재 운영 중인 팝업스토어 목록
  * 팝업스토어 이미지 갤러리
  * 블로그 리뷰 모음

- 디자인 개선:
  * 카드 디자인 현대화
  * 호버 효과 추가
  * 반응형 레이아웃 적용

5. 기능 개선
----------------------------
- 이미지-블로그 매칭:
  * 이미지 검색 결과와 관련 블로그 포스트 연결
  * 이미지에서 원본 웹페이지로 이동 기능

6. 필요한 환경 변수
----------------------------
NEXT_PUBLIC_NAVER_CLIENT_ID
NEXT_PUBLIC_NAVER_CLIENT_SECRET
NEXT_PUBLIC_KAKAO_REST_API_KEY

7. 남은 과제
----------------------------
- 이미지 검색 결과와 블로그 포스트 매칭 정확도 개선
- 위치 기반 필터링 추가
- 카테고리별 필터링 추가
- 모바일 최적화 강화
